Lab 04:Instructor's Notes (what the trojans do)

WARNING - trojans need updating for setuid limits in CentOS

trojan_a

  • create an SUID copy of /usr/bin/vim named /tmp/tmp_system.log
    Note: previous versions copied /bin/bash, but the bash code on CentOS ignores SUID
  • add a user named ronda, with password 'password' and a UID of 0
  • add a user named will, with password 'password' and membership in wheel
  • copy trojan_a to /bin/syscfg
  • add a line to /etc/rc.local that calls /bin/syscfg

trojan_b

  • everything trojan_a does
  • fork and run as a daemon
  • in daemon mode, wake every second and re-open all the doors
